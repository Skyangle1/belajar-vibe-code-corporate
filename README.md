## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Skyangle1/belajar-vibe-code-corporate.git

# Navigate to the project
cd belajar-vibe-code-corporate

# Install dependencies
npm install
```

### Development

```bash
# Start development server
npm run dev

# Build for production
npm run build

# Run tests
npm test
```
